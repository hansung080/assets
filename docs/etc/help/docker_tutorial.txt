###### docker tutorial ######

### docker install ###
# requirement
1. Kernel version need to be equal or higher than 3.10.
2. Linux distribution need to be supportable.
3. Linux need to be 64-bit.
4. Installation requires root permission.

$ uname -a
Linux ubuntu64 4.15.0-43-generic #46-Ubuntu SMP Thu Dec 6 14:45:28 UTC 2018 x86_64 x86_64 x86_64 GNU/Linux

$ uname -r
4.15.0-43-generic

$ sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
$ sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"
$ sudo apt-get update
$ sudo apt-get install docker-ce
